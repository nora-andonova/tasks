What does the following function do?

```python
class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

def func(head):
    slow, fast = head, head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            return True
    return False

```
